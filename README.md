# 新座市 緑地管理システム

新座市内の緑地（公園・緑地）を地図上で管理するためのシンプルなWebアプリです。
ブラウザだけで動作し、サーバーは不要です（データはブラウザのlocalStorageに保存されます）。

## 使い方

[GitHub Pages で開く](https://REPLACE_ME.github.io/niiza-ryokuchi-kanri/) ※公開後にURLが決まります

または `index.html` をブラウザで直接開いても使えます。

## 機能

- 🗺️ 地図（OpenStreetMap）をクリックして緑地を新規登録
- 🏷️ 保全作業の種別をプルダウンで自由に追加・編集（草刈り・剪定・除草など）
- 📅 年度ごとに保全作業の記録を追加
  - 実施日・作業種別・実施業者・費用
  - 入力者・発注担当者
  - 進捗状況（見積もり依頼中・見積もり済み・発注済み・施工中・施工完了など、編集可能）
- 💰 年度別予算を設定すると、支出合計・残額を自動計算（予算超過時は警告表示）
- 📷 記録ごとに作業前の写真をアップロード・確認
- 📄 緑地ごとにPDF敷地図をアップロードし、その上に位置をプロット（複数ピン・ラベル付き）
- ✏️ 記録の編集・削除、緑地の地図上位置の再設定

## データの保存について

すべてのデータはブラウザのlocalStorageに保存されます。サーバーには送信されません。
複数人・複数端末でデータを共有する場合は、別途バックエンド（データベース）化が必要です。

## 技術スタック

- 素のHTML / CSS / JavaScript（フレームワーク不使用）
- [Leaflet](https://leafletjs.com/) + OpenStreetMap（地図表示）
- [PDF.js](https://mozilla.github.io/pdf.js/)（PDF表示）
